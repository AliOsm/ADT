# ADT
[![All Contributors](https://img.shields.io/badge/all_contributors-2-orange.svg?style=flat-square)](#contributors)

Standard implementation of **A**lgorithms, **D**ata structures and **T**echniques.

This repository was created to gather the standard implementation for the most known algorithms in different programming languages.

## Implemented Stuffs

```
ADT
├── Algorithms
│   ├── Graph
│   │   ├── Breadth First Search
│   │   ├── Depth First Search
│   │   ├── Dijkstra
│   │   ├── Floyd-Warshall
│   │   ├── Prim
│   │   ├── Kruskal
│   │   └── Topological Sort
│   ├── Math
│   │   ├── Fast Power
│   │   ├── n Choose k
│   │   └── Number Theory
│   │       ├── N'th Catalan Number
│   │       ├── Greatest Common Divisor
│   │       ├── Least Common Multiple 
│   │       └── Prime Numbers
│   │           └── Sieve of Eratosthenes
│   └── Sort
│       ├── Insertion Sort
│       ├── Merge Sort
│       ├── Heap Sort
│       ├── Bubble Sort
│       ├── Counting Sort
│       ├── Quick Sort
│       └── Selection Sort
├── Data Structures
│   ├── Doubly Linked List
│   ├── Linked List
│   └── Trie
└── Techniques
    └── Dynamic Programming
        └── Longest Increasing Subsequence
 ```
